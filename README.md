# SQL Employee Data Analysis Project

##  Project Overview
This project is created to practice SQL for Data Analyst roles.
It contains an employee dataset and SQL queries for data analysis.

##  Dataset Information
**Table Name:** employees  

Columns:
- emp_id (Employee ID)
- name
- department
- salary
- city
- joining_date

##  Tools Used
- SQL (MySQL / PostgreSQL / SQL Server)
- GitHub

##  SQL Concepts Practiced
- SELECT
- WHERE
- ORDER BY
- GROUP BY
- HAVING
- Aggregate Functions (SUM, AVG, COUNT)

##  Example Analysis
- Average salary by department
- Employee count by city
- Highest and lowest salaries
- Employees joined after 2021
